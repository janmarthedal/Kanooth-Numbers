Sputsoft Numbers is a C++ library for doing multiple-precision arithmetic. The underlying theory and design can be found among the multiple-precision articles.

The library is header-only and portable, relying on some boost libraries (also header-only). It is possible, however, to obtain high-performance code by choosing the GMP backend. This requires no code changes, but only setting a preprocessor macro and linking to the GMP library.
